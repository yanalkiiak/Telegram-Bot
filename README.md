# Telegram Bot

This is a Telegram bot project built with Python using the aiogram framework.
The project was created as part of my learning process in backend development.

The bot processes user messages, works with custom keyboards, and is structured using routers for better code organization.

## What I implemented

- Message handling with aiogram routers  
- Custom keyboards for user interaction  
- Command handling  
- Modular project structure (handlers, keyboards)  

## Technologies

- Python  
- aiogram  
- asyncio  

## How to run

1. Clone the repository:

git clone https://github.com/yanalkiiak/Telegram-Bot.git


2. Install dependencies:

pip install -r requirements.txt


3. Create a `.env` file:

BOT_TOKEN=your_token_here


4. Run the bot:

python main.py
